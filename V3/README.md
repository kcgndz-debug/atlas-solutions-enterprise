# Atlas Enterprise V3 Full Preview

This is a parallel preview build. Keep Atlas V2.1.1 unchanged until this version is tested and approved.

Included:
- Unlimited estimate line items
- Add, duplicate, delete, move up, and move down
- Rail, fence, and custom items
- Travel & Mobilization applied once per estimate
- Round-trip mileage, field verification, travel labor, hotel, per diem, tolls, parking, miscellaneous, and manual override
- Google Maps navigation button
- Material Request workflow
- Dashboard alerts and open request counts
- Customers, projects, documents, field reports, and Athena shell
- Local/offline storage
- Print/PDF-ready estimator
- Installable PWA shell

Recommended:
1. Upload to the existing feature branch or a new preview branch.
2. Deploy through a Netlify preview.
3. Test before merging to main.
