ATLAS COMMAND CENTER v3

DEPLOY TO NETLIFY
1. Unzip this package.
2. Upload the entire Atlas_Command_Center_v3 folder to Netlify Drop.
3. Do not upload index.html by itself.
4. After deployment, open the site in Safari and use Add to Home Screen.
